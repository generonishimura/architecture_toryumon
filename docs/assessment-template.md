# 設計力セルフアセスメント

> 本テンプレートは、ワークショップの開始時（事前）と終了時（事後）に同じ項目で自己評価を行い、学習の成長を可視化するためのものです。

---

## 使い方

1. **オリエンテーション（第0回）** で事前評価を実施
2. **最終回（第10回）** で事後評価を実施
3. 事前・事後の差分を確認し、成長を振り返る

### 評価スケール

| レベル | 意味 |
|--------|------|
| 1 | 聞いたことがない / まったく分からない |
| 2 | 用語は知っているが、説明できない |
| 3 | 概念を説明できるが、実務で使ったことがない |
| 4 | 実務で使ったことがあり、基本的な判断ができる |
| 5 | 他の人に教えられる / トレードオフを踏まえた判断ができる |

## 振り返りガイド（最終回で使用）

### スコアの変化を確認

事前・事後のスコア差が大きい項目 → ワークショップで特に成長した領域

事後でも3以下の項目 → 今後の自学自習の重点テーマ

### 振り返りの問い

1. **最も成長を感じた領域はどこですか？その理由は？**
2. **最も理解が難しかった領域はどこですか？何が障壁でしたか？**
3. **明日の業務で最初に活かしたい学びは何ですか？**
4. **今後、さらに深く学びたいテーマは何ですか？**

### チームでの共有（推奨）

- 各自の振り返り結果をチーム内で共有する
- チーム全体で弱い領域があれば、フォローアップの勉強会を検討する
- 成長が大きかった参加者の学び方を共有し、チームの学習文化を醸成する
