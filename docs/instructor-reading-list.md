# 参考書籍クイックリファレンス（講師向け）

> 限られた準備時間で最大限の効果を得るため、各書籍の「講師として押さえるべき章」を厳選した。通読が理想だが、優先章だけでも目を通すことで講義の質を担保できる。

## 凡例

- 必読: 担当回の講師は必ず該当章を読了すること
- 推奨: 時間が許せば読んでおくと講義に深みが出る
- 参考: 受講者からの質問対応や補足説明に役立つ

## 読書の優先順位ガイド

準備時間に応じた読書戦略を以下に示す。

- **最低限（各回の前日）**: 担当回の「必読」書籍の優先章のみ読む（3-4時間）
- **標準（1週間前）**: 「必読」書籍の優先章に加え、「推奨」書籍の優先章にも目を通す
- **理想（1ヶ月前）**: 「必読」書籍を通読し、「推奨」「参考」書籍の優先章も読了する

全フェーズを通じて最初に読むべき3冊は、5番（成瀬本）、7番（Clean Architecture）、2番（達人に学ぶDB設計）である。この3冊の優先章を押さえるだけで、講義全体の骨格を理解できる。
