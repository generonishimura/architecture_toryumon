# 回をまたぐ「つなぎ台本」

> 各回の講師が異なる場合でも、参加者にとって学びが一本の線としてつながるように、回の冒頭と終了時に使う「つなぎ台本」をまとめたものです。講師間の引き継ぎコストを下げ、参加者が「前回やったことと今日やることの関係」を自然に理解できる状態をつくることを目的としています。

## 使い方

- **冒頭の振り返り台本（A）** は、その回の講師が開始5分間で使います。前回の講師から引き継ぎがなくても、この台本を読めば最低限の接続ができます。
- **終了時の橋渡し台本（B）** は、その回の講師がクロージングの2〜3分で使います。次回の講師が誰であっても、参加者が「次に何を学ぶか」の期待値を持てるようにします。
- 台本中の「接続フレーズ」「次回への接続フレーズ」は、そのまま口に出して読める文言にしてあります。自分の言葉に変えても構いませんが、趣旨は保ってください。
